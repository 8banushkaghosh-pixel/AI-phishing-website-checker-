About This Project

The AI Phishing Website Checker helps users identify potentially unsafe websites by analyzing URLs for common phishing indicators. It provides a risk score and security recommendations to promote safer internet browsing.

Cybersecurity Tips

  • ✔ Always check if the website uses HTTPS.
  • ✔ Never enter passwords on suspicious websites.
  • ✔ Verify the website URL before logging in.
  • ✔ Avoid clicking unknown email links.
  • ✔ Keep your browser and antivirus updated.
